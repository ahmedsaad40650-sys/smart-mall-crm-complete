## 📋 نظرة عامة

**Smart Mall CRM Park St.** هو الحل الشامل والمتكامل لإدارة المولات التجارية الكبرى. في هذا الإصدار (v3.0.0)، انتقلنا من مجرد CRM إلى **"نظام تشغيل ذكي" (Smart Mall OS)** من خلال دمج محرك **NCRI (Neural CRM Intelligence)**. 

### 🎯 التحديثات الجديدة في v3.0.0
- 🧠 **محرك NCRI الذكي**: ذكاء اصطناعي يحلل البيانات ويتخذ قرارات آلية (إنشاء طلبات صيانة، تسجيل مخالفات، تقييم المستأجرين).
- ⚡ **Auto-Execution Engine**: أتمتة كاملة لدورة العمل، المساعد الذكي لا يقترح فقط، بل ينفذ العمليات فوراً.
- 📊 **تحليل ذكي للوحدات (Module Analysis)**: كل قسم (صيانة، تسويق، مبيعات) مدعوم بمحرك AI مخصص لتحليل المشاكل وإصدار قرارات وتوصيات لحظية.
- ✅ **مركز التقارير المطور**: دعم التقارير الضخمة، التحميل الخلفي، وصيغ متعددة (PDF, Excel).
- ✅ **بوابة العملاء والمستأجرين**: واجهات مخصصة للعملاء لعرض الخدمات والفعاليات ودليل المول באופן راقٍ.
- ✅ **نظام الصلاحيات الدقيق (RBAC Pro)**: أمان متطور يعزل العمليات الإدارية عن المستأجرين والعملاء.

## 🏗️ البنية التقنية

### Backend
- **Framework**: Flask 2.3.3 (Python)
- **Database**: MongoDB 7.0 (Data Storage)
- **Authentication**: JWT & RBAC 2.0 (Security)
- **Reporting**: Advanced Reports Engine (JSON/PDF/Excel)

### Frontend
- **Framework**: React 18 + Vite 4
- **State Management**: Zustand
- **UI & Styling**: Tailwind CSS 4 + Lucide Icons
- **Visualization**: Recharts & Chart.js
- **Deployment**: Firebase Hosting / Cloud Functions

### DevOps & Tools
- **Containerization**: Docker & Docker Compose
- **Cloud Infrastructure**: Google Firebase
- **Automation**: GitHub Actions / Bash Deploy Scripts
- **Management Tools**: Standalone RBAC & Interactive Demos

## 📦 التثبيت

### المتطلبات

- Docker 20.10+
- Docker Compose 2.0+
- 4GB RAM
- 10GB مساحة تخزين

### التثبيت السريع

```bash

git clone https://github.com/your-repo/smart-mall-crm.git
cd smart-mall-crm

# 2. تشغيل سكريبت الإعداد التلقائي
./scripts/setup.sh
```

```bash

cp backend/.env.example backend/.env

# 2. بناء وتشغيل الحاويات
cd docker
docker-compose build
docker-compose up -d

# 3. التحقق من الحالة
docker-compose ps
```

## 🔧 إدارة النظام

### الأوامر الأساسية

```bash
# عرض سجلات النظام
docker-compose logs -f

# إيقاف النظام
docker-compose down

# إعادة تشغيل النظام
docker-compose restart

# إعادة بناء الحاويات
docker-compose build --no-cache

# تنظيف النظام بالكامل
docker-compose down -v
```

### النسخ الاحتياطي

```bash
# نسخ قاعدة البيانات
docker exec smartmall_mongodb mongodump --out /backup

# استعادة قاعدة البيانات
docker exec smartmall_mongodb mongorestore /backup
```

## 🔌 API Documentation

### المصادقة

```bash
# تسجيل الدخول
POST /api/auth/login
Content-Type: application/json

{
  "email": "admin@smartmall.com",
  "password": "admin123"
}

Response:
{
  "access_token": "eyJ0eXAiOiJKV1QiLCJhbGc...",
  "user": {
    "id": "user_123",
    "name": "مدير النظام",
    "role": "admin"
  }
}
```

### الوحدات التجارية

```bash
# الحصول على جميع الوحدات
GET /api/units

# إنشاء وحدة جديدة
POST /api/units
Authorization: Bearer {token}
Content-Type: application/json

{
  "unit_number": "A-101",
  "floor": "1",
  "area": 150,
  "rental_price": 5000,
  "type": "retail",
  "status": "available"
}
```
